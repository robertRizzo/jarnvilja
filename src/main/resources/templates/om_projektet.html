Om Projektet

Järnvilja är ett fullstack-portfolioprojekt byggt för att demonstrera moderna webbutvecklingstekniker i ett realistiskt scenario – en kampsportklubb med bokning, autentisering och administrationsfunktioner.

Teknikstack

Backend

  • Java 17
  • Spring Boot 3.4
  • Spring Security (rollbaserad åtkomstkontroll)
  • Spring Data JPA / Hibernate
  • Lombok
  • Maven

Frontend

  • Thymeleaf (server-side rendering)
  • HTML5 semantiska element
  • CSS3 (dark theme, responsiv design)
  • Vanilla JavaScript (ES6+)

Databas

  • MySQL 8 (produktion)
  • H2 In-Memory (demo/test)
  • Spring Profiles (dev/demo/prod)

DevOps & Verktyg

  • Docker & Docker Compose
  • GitHub Actions CI
  • Git versionshantering

Arkitektur

Presentationslager

Thymeleaf-templates med fragment-system för layout-återanvändning. Responsiv CSS med dark theme.

Controllerlager

Separata controllers per roll (Member, Admin, Trainer) med @PreAuthorize för säkerhet.

Servicelager

Affärslogik med @Transactional för dataintegritet och @Service-beans.

Datalager

Spring Data JPA-repositories med custom queries, indexering och unika constraints.

Demonstrerade kompetenser

REST API-design Rollbaserad åtkomstkontroll CSRF-skydd Databasdesign & normalisering Transaction management Global exception handling DTO-mönster Spring Profiles Docker-containerisering CI/CD-pipeline Responsiv design Tillgänglighet (ARIA) Database seeding E-postintegration Input-validering
